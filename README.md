# Книга контактов - React приложение

Простое React-приложение для управления списком контактов.

# Функциональность

- Добавление контактов (имя и телефон)
- Удаление контактов
- Сохранение данных в LocalStorage
- Валидация полей ввода

# Технологии
- React
- JavaScript
- СSS
- LocalStorage
